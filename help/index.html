A) HTML Structure - top

This template is a responsive layout with two columns. The HTML structure is basically divided in four parts:

  • Modal: Here comes the content for the signin/signup modal form.
  • Navbar: Here comes the content for the fixed navbar, the multilevel dropdowns, the search form and the signin/signup button.
  • Left Content: Here comes the main content. News slider and carrousel, latest news boxes, post content, comments and comment form.
  • Sidebar: Here comes all the widgets that the user may want to add to the theme.
  • Footer: Here comes the secondary nav, scroll to top button, three columns for widgets, credits and social networks.

B) CSS Files and Structure - top

The theme uses three CSS files.

style.css contains all the styling of the site, structure, colors and more. All styling changes must be done here.

  • 0. Document Setup
  • 1. Set Categoty Colors
  • 2. Navigation & Menus
  • 3. Main Content Styles
  • 4. Post Content Styles
  • 5. Comments
  • 6. Category Styles
  • 7. Page Templates
  • 8. Widgets
  • 9. Footer Styles
  • 10. Media Queries

bootstrap.css contains all the core styles from the Bootstrap library. It´s not recomended to change anything on this stylesheet.

font-awesome.min.css contains the styles for the extended glyphicons from Font Awesome.

C) JavaScript - top

This theme imports nine Javascript files:

  • jquery.min.js
  • jquery.mobile.min.js
  • bootstrap.min.js
  • jquery.imagesloaded.min.js
  • jquery.masonry.min.js
  • jquery.fitvids.js
  • jquery.custom.js
  • jquery.placeholder.js
  • jquery.validate.min.js

jquery.min.js is a fast and concise JavaScript Library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development.

jquery.mobile.min.js is a HTML5-based user interface system designed to make responsive web sites and apps that are accessible on all smartphone, tablet and desktop devices. In this theme is mainly used to create the touch-slide fuctionality on the slider.

bootstrap.min.js contains all the UI elements that Bootstrap offers, tabs, modal, tooltip, corrousel.

jquery.imagesloaded.min.js is a small plugin for jQuery that detect when images have been loaded. It's specially important for the masonry plugin.

jquery.masonry.min.js is a JavaScript grid layout library. It works by placing elements in optimal position based on available vertical space, sort of like a mason fitting stones in a wall.

jquery.fitvids.js is a lightweight, easy-to-use jQuery plugin for fluid width video embeds.

jquery.custom.js contains the trigger for the tabs, tooltip, scrollable, the ajax post load, the dropdowns and the mobile functionality.

jquery.placeholder.js is a jQuery plugin that enables HTML5 placeholder behavior for browsers that aren’t trying hard enough yet.

jquery.validate.min.js is a jQuery plugin that makes simple clientside form validation easy, whilst still offering plenty of customization options.
